PowerShell

Keep computer screen alive
                        
                            Write-Host "Screen will not turn off"
                            $wsh = New-Object -ComObject WScript.Shell
                            while ($true) {
                            $wsh.SendKeys("{NUMLOCK}")
                            Write-Host "Screen will never turn off"
                            Start-Sleep -Seconds 30
                            }

SendKeys("{NUMLOCK}") can be actual number"{1}"

Start-Sleep -Seconds 30: pauses the script for 30 seconds
